# UniList

This is a website that gives you information on all universities in the world while also giving information on continents and countries

## Tools

<ul>
  <li>React</li>
  <li>Redux</li>
  <li>React Hooks</li>
  <li>GraphQL (Apollo Client)</li>
  <li>Bootstrap</li>
</ul>
